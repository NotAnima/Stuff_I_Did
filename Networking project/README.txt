The sensitivity for the PIR are set to minimum sensitivity and minimum delay. Sensitivity at min according to the docs is still 3m detection range and the delay doesn't matter because we are doing the delay in code anyway. Minimum setting is: As anti-clockwise as possible.

To install dependencies: download the requirements.txt and run

pip install -r requirements.txt

in the correct folder with the file in it

FURTHER DEPENDENCIES:
1) sudo apt update 
2) sudo apt install fswebcam
3) sudo apt install ffmpeg
